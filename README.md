# ☕ 카페 클리커

커피를 클릭해서 코인을 모으고, 업그레이드로 매장을 확장하는 클리커 게임입니다.

## 플레이 방법

1. 화면 중앙의 커피컵을 클릭해 코인을 획득합니다.
2. 오른쪽 업그레이드 패널에서 장비와 인력을 구매합니다.
3. 업그레이드할수록 클릭당 수입과 초당 자동 수입이 증가합니다.
4. 누적 수입에 따라 카페 칭호가 올라갑니다.

## 업그레이드 목록

| 이름 | 효과 | 기본 비용 |
|------|------|-----------|
| 🫘 좋은 원두 | 클릭당 +1원 | 15원 |
| ⚙️ 에스프레소 머신 | 초당 +2원 | 50원 |
| 👩‍🍳 바리스타 고용 | 초당 +10원 | 200원 |
| 🪴 인테리어 개선 | 클릭당 +5원 | 400원 |
| 🛵 배달 서비스 | 초당 +25원 | 1,000원 |
| 🏪 2호점 오픈 | 초당 +100원 | 5,000원 |
| 🏢 프랜차이즈 | 초당 +500원 | 25,000원 |
| 📱 앱 출시 | 클릭당 +50원 | 10,000원 |

> 같은 업그레이드를 반복 구매할수록 가격이 1.6배씩 상승합니다.

## 기술 스택

- Vanilla HTML / CSS / JavaScript (외부 라이브러리 없음)
- 정적 파일 단 1개로 구성 (`index.html`)

## 배포

### Vercel (권장)

```bash
npm i -g vercel
vercel --prod
```

또는 [vercel.com/new](https://vercel.com/new) 에서 zip 파일을 드래그 앤 드롭으로 업로드합니다.

### 로컬 실행

별도 서버 없이 `index.html`을 브라우저에서 바로 열 수 있습니다.

```bash
open index.html   # macOS
start index.html  # Windows
```
